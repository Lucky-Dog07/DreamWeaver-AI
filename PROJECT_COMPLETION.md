## 🚀 如何使用

### 快速启动

#### Windows
```bash
# 双击运行
run.bat
```

#### macOS/Linux
```bash
bash run.sh
```

### 手动启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置API密钥
cp .env.example .env
# 编辑 .env 文件

# 3. 检查配置
python check_setup.py

# 4. 启动应用
streamlit run src/app.py
```

### 应用使用流程

1. **首页** → 选择"智能画板"或"作品工坊"
2. **智能画板** → 绘画 → 小精灵分析 → 查看结果
3. **作品工坊** → 上传图片 → 生成音乐/视频 → 下载结果
4. **设置** → 自定义界面和功能

## 🔧 技术栈

### 前端
- **框架**: Streamlit 1.30+
- **绘画**: HTML5 Canvas
- **样式**: CSS3 + Streamlit内置主题

### AI/ML
- **视觉理解**: Qwen-VL-Chat (阿里云DashScope)
- **文本理解**: Qwen3-Omini-Flash
- **语音合成**: TTS (Qwen3-Omini-Flash)
- **音乐生成**: Coze工作流
- **视频生成**: 火山引擎Seedance

### 后端
- **语言**: Python 3.8+
- **数据处理**: NumPy, Pillow, OpenCV
- **音频处理**: Wave, SciPy
- **HTTP**: Requests

### 存储
- **本地存储**: 文件系统
- **数据格式**: JSON, PNG, WAV, MP3

## ⚠️ 已知限制和未来改进

### 当前限制
1. **画板清晰度**: Canvas分辨率固定为800x600，可扩展
2. **离线模式**: 目前需要互联网连接，未来可支持本地模型
3. **实时协作**: 暂不支持多人实时协作
4. **复杂动画**: 视频生成基于首尾帧，支持有限

### 未来改进方向
1. **WebGL支持** - 支持更大分辨率和复杂笔刷
2. **离线模型** - 集成本地小模型支持离线使用
3. **实时协作** - 支持家长和老师实时查看
4. **AR/VR支持** - 支持沉浸式体验
5. **多语言支持** - 支持多种语言界面和语音

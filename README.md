# DreamWeaver AI - 绘梦精灵 🧚

让每个孩子的画都能被看见、被听见、被记住。

## 功能特色

- 🎨 **智能画板** - 自由绘画，小精灵实时陪伴与反馈
- 🎵 **音乐生成** - 根据画作生成专属背景音乐
- 🎬 **视频生成** - 让静态画作动起来
- 🧚 **AI伙伴** - 小精灵球球全程陪伴鼓励

## 技术栈

- Streamlit - Web应用框架
- DashScope - 阿里云多模态AI
- Coze - 字节跳动AI平台
- OpenCV - 图像处理
